# 📊 Project 2 - Data Classification Using AI

## 📖 Overview
This project demonstrates a basic supervised machine learning classification model developed as part of the Decode Labs AI Internship.

## 🎯 Objective
Predict whether a student will pass or fail based on:
- Hours Studied
- Attendance Percentage

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn

## 📚 Machine Learning Concepts
- Dataset Loading
- Feature Selection
- Train-Test Split
- Decision Tree Classifier
- Model Evaluation
- Prediction

## ▶️ How to Run

```bash
pip install -r requirements.txt
python classification.py
```
